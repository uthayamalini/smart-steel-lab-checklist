# smart-steel-lab — Version 2.0  
**AI-powered Steel Testing & Selection Assistant**

---

## 🚀 Overview
Smart Steel Lab is a modern web application that helps engineers, builders, students, and manufacturers choose and validate steel grades based on their application needs.

Version 2.0 introduces a powerful AI-driven test plan generator, giving users expert-level guidance instantly.

**Live demo:**  
👉 https://steellab.vercel.app/

---

## 🎯 Features
- AI-powered steel test recommendations  
- Smart checklist engine for grade + application  
- Standards-aware reasoning (AI explains why each test is required)  
- Minimalist, fast UI built with Next.js  
- Mobile-friendly for labs, factories, and field use  

---

## 🧠 Tech Stack
- Next.js 14 (App Router)  
- TypeScript  
- TailwindCSS  
- Llama-based AI reasoning (Groq Llama 3.3)  
- Vercel Deployment
